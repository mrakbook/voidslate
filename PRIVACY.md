# Privacy Policy for VoidSlate

Last updated: May 31, 2026

VoidSlate is a Visual Studio Code color theme created by **MrakBook - Boris Karaoglanov**.

## Summary

VoidSlate is intended to be a theme-only extension. It does not intentionally collect, store, sell, transmit, or process personal data.

## Data collection

VoidSlate does not intentionally collect:

- Personal information.
- Usage analytics.
- Telemetry.
- Location data.
- Source code.
- File contents.
- Account information.

## Network requests

VoidSlate is not intended to make network requests.

## Marketplace and platform data

Visual Studio Code, the Visual Studio Marketplace, GitHub, or other distribution platforms may process installation statistics, ratings, reviews, crash data, account data, or other platform data according to their own policies. That platform data is not controlled by VoidSlate.

## Changes to this policy

If VoidSlate adds features that collect or transmit data, this privacy policy must be updated before those features are released.
