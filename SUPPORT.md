## Before asking for help

Please include:

- Your Visual Studio Code version.
- Your operating system.
- The VoidSlate version, currently `0.0.1` for this package state.
- A screenshot if the issue is visual.
- Steps to reproduce the issue.
- Whether the issue happens with all files or only a specific language/file type.

## GitHub issues

Use GitHub issues for:

- Theme color bugs.
- Accessibility/readability problems.
- Syntax highlighting issues.
- Marketplace listing issues.
- Documentation corrections.

Do not use public issues to request confidential commercial licensing terms. For commercial-use permission, email [boris@mrakbook.com](mailto:boris@mrakbook.com).
