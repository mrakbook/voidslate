# Security Policy for VoidSlate

## Supported versions

Security support is provided for VoidSlate `0.0.1`.

| Version | Supported |
| --- | --- |
| 0.0.1 | Yes |
| Older releases | Best effort |

## Reporting a vulnerability

If you find a security issue, do not publish exploit details in a public issue first.

Email: [boris@mrakbook.com](mailto:boris@mrakbook.com)

Please include:

- A clear description of the issue.
- Steps to reproduce the issue.
- The affected version or commit.
- Any relevant screenshots or logs.
- Whether the issue involves dependencies, packaging, Marketplace publishing, or extension behavior.

## Theme-only privacy and security note

VoidSlate is intended to be a theme-only Visual Studio Code extension. It should not require network access, telemetry, analytics, or external services. If future code adds those behaviors, the privacy policy and security documentation must be updated before release.
