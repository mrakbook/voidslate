# Legal Notes for VoidSlate

These notes are practical project documentation and are not a substitute for advice from a qualified attorney.

## Licensing position

VoidSlate is intended to be:

- Free to use for non-commercial purposes.
- Free to modify for non-commercial purposes.
- Free to share for non-commercial purposes.
- Not permitted for commercial use without prior written permission from MrakBook - Boris Karaoglanov.

## Why PolyForm Noncommercial License 1.0.0 is used

VoidSlate is software, so a software-focused non-commercial license is used instead of a Creative Commons license for the theme code and extension files.

## Open source wording

Because VoidSlate restricts commercial use, use wording like:

- "source-available"
- "free for non-commercial use"
- "non-commercial license"

Avoid claiming that VoidSlate is OSI-approved open source unless the license changes to an OSI-approved license.
